# 𝕏 (Twitter) Thumbnail Simulator

X(Twitter)のタイムラインで画像がどのように表示されるかをシミュレートするツールです。

## 🌟 特徴

- **完全フロントエンド**: GitHub Pagesで動作する軽量なWebアプリケーション
- **3つの画像追加方法**:
  - ドラッグ&ドロップ
  - ファイル選択ダイアログ
  - クリップボードからの貼り付け（Ctrl+V / Cmd+V）
- **リアルなプレビュー**: Xのタイムラインと同じレイアウトでサムネイルを表示
- **最大4枚まで対応**: Xの仕様に準拠

## 📱 サムネイル表示ルール

Xのサムネイル表示ロジックを再現しています：

- **1枚（縦長）**: アスペクト比 3:4 (padding-bottom: 133.333%)
- **1枚（横長）**: アスペクト比 16:9 (padding-bottom: 56.25%)
- **2枚**: 横に並べて表示
- **3枚**: 左に1枚、右に2枚縦並び
- **4枚**: 2x2のグリッド表示

## 🚀 使い方

1. 画像をアップロード:
   - エリアに画像をドラッグ&ドロップ
   - 「ファイルを選択」ボタンをクリック
   - Ctrl+V（またはCmd+V）でクリップボードから貼り付け

2. プレビューを確認:
   - 追加した画像がXのタイムライン形式で表示されます

3. クリア:
   - 「クリア」ボタンで全ての画像を削除

## 🛠️ 技術スタック

- **HTML5**: セマンティックなマークアップ

- **Vanilla JavaScript**: フレームワークなしの軽量実装
- **GitHub Pages**: 静的サイトホスティング

## 📦 ローカルでの実行

```bash
# リポジトリをクローン
git clone https://github.com/Kuyoh-ai/XThumbTester.git

# ディレクトリに移動
cd XThumbTester

# ローカルサーバーを起動（例: Python）
python -m http.server 8000
# または
python3 -m http.server 8000

# ブラウザで開く
# http://localhost:8000
```
